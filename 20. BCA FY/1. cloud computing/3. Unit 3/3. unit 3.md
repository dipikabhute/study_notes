1. Introduction to Cloud Computing
What is Cloud Computing?

Cloud Computing is the delivery of computing services like servers, storage, databases, networking, software, analytics and intelligence over the internet.

Example
Google Drive (storage)
Gmail (software)
AWS (servers)
Microsoft Azure (cloud platform)
2. Characteristics of Cloud Computing
1. On-Demand Self Service

Users can access resources automatically without human interaction.

Example:
You create a virtual machine in AWS instantly.

2. Broad Network Access

Services are available over internet using mobile, laptop, tablet.

Example:
Google Drive access from mobile & laptop

3. Resource Pooling

Resources shared among multiple users.

Example:
Multiple companies using same AWS servers

4. Rapid Elasticity

Resources increase or decrease automatically.

Example:
Website traffic increases → cloud adds servers automatically

5. Measured Service

Pay only for what you use.

Example:
Pay only for used storage

3. Benefits of Cloud Computing
1. Cost Saving

No need to buy hardware

2. Scalability

Increase resources anytime

3. Accessibility

Access from anywhere

4. Automatic Updates

No manual update required

5. Backup & Recovery

Automatic data backup

4. Advantages of Cloud Computing

✔ Low cost
✔ High scalability
✔ Easy access
✔ Data backup
✔ Automatic updates
✔ Better collaboration
✔ No maintenance

5. Disadvantages of Cloud Computing

✖ Internet dependency
✖ Security issues
✖ Limited control
✖ Downtime risk
✖ Data privacy concerns
