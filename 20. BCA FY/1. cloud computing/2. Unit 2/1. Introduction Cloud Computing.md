# ☁️ **Introduction to Cloud Computing**

### ✅ **Definition**

Cloud Computing is a technology that allows users to access data, software, and services over the internet instead of storing them on local computers.

👉 Example: Google Drive, Dropbox, AWS

---

# ⭐ **Characteristics of Cloud Computing**

1. **On-demand self-service**
   Users can access resources anytime without human interaction.

2. **Broad network access**
   Services are available over the internet (mobile, laptop, etc.)

3. **Resource pooling**
   Multiple users share the same resources (multi-tenant model)

4. **Rapid elasticity**
   Resources can increase or decrease quickly as needed

5. **Measured service**
   Pay only for what you use (pay-as-you-go)

---

# 🎯 **Benefits of Cloud Computing**

* Cost saving (no need for hardware)
* Easy access from anywhere
* High scalability
* Automatic updates
* Backup & recovery

---

# 👍 **Advantages of Cloud Computing**

* Low cost (no infrastructure setup)
* High availability
* Easy data backup
* Flexibility & scalability
* Supports remote work

---

# 👎 **Disadvantages of Cloud Computing**

* Internet dependency
* Security & privacy issues
* Downtime risk
* Limited control over data
* Vendor lock-in

---

# 📜 **History of Cloud Computing**

* **1960s** → Concept started (John McCarthy idea)
* **1990s** → Internet growth
* **2000s** → Modern cloud started
* **2006** → AWS launched cloud services
* Today → Widely used in IT industry

---

# 🏗️ **Cloud Architecture (Cloud Reference Model)**

### 🔹 Frontend

* User interface (browser, apps)

### 🔹 Backend

* Servers
* Storage
* Databases
* Virtual machines

### 🔹 Middleware

* Software that connects frontend & backend

👉 All components work together to deliver cloud services.

---

# ☁️ **Service Models**

## 1. Infrastructure as a Service (IaaS)

* Provides virtual hardware (servers, storage)
* User manages OS & apps
  👉 Example: AWS EC2

---

## 2. Platform as a Service (PaaS)

* Provides platform to develop apps
* No need to manage hardware
  👉 Example: Google App Engine

---

## 3. Software as a Service (SaaS)

* Software available via internet
* No installation needed
  👉 Example: Gmail, MS Office 365

---

# 🌍 **Types of Cloud**

1. **Public Cloud**

   * Open to everyone
   * Example: AWS, Azure

2. **Private Cloud**

   * Used by single organization

3. **Hybrid Cloud**

   * Combination of public + private

4. **Community Cloud**

   * Shared by multiple organizations

---

# ⚙️ **Technologies for Web Services Deployment**

* Virtualization
* Web services (SOAP, REST)
* APIs
* Containers (Docker)
* Microservices architecture

---

# 🚀 **Deploying Web Services**

## 🔹 Inside Cloud

* Hosted within cloud infrastructure
* High scalability
* Better performance

### Advantages:

* Fast deployment
* Auto scaling
* Reliable

### Disadvantages:

* Cost may increase
* Vendor dependency

---

## 🔹 Outside Cloud (On-Premise)

* Hosted on local servers

### Advantages:

* Full control
* Better security (in some cases)

### Disadvantages:

* High cost
* Limited scalability
* Maintenance required

---
