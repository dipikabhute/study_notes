## ❓ Q1. What is Virtualization?

**Answer:**
Virtualization is the process of creating a virtual version of physical resources such as servers, storage, or networks using software called a hypervisor. It allows multiple operating systems to run on a single physical machine.

---

## ❓ Q2. Define Hypervisor.

**Answer:**
A hypervisor is software that creates and manages virtual machines by dividing physical hardware resources among multiple operating systems.

👉 Example: VMware ESXi

---

## ❓ Q3. What is Cloud Computing?

**Answer:**
Cloud computing is the delivery of computing services like servers, storage, databases, and software over the internet on a pay-as-you-go basis.

---

## ❓ Q4. What are types of virtualization?

**Answer:**

* Full Virtualization
* Para-Virtualization
* OS-Level Virtualization
* Hardware-Assisted Virtualization

---

## ❓ Q5. What is IaaS?

**Answer:**
Infrastructure as a Service (IaaS) provides virtualized computing resources like servers, storage, and networking over the internet.

---

# 📘 SECTION B: Medium Answer Questions (5–8 Marks)

## ❓ Q6. Explain Taxonomy of Virtualization Techniques.

**Answer:**
Taxonomy refers to classification of virtualization techniques:

1. **Full Virtualization** – Complete simulation of hardware
2. **Para-Virtualization** – OS modified for better performance
3. **OS-Level Virtualization** – Containers share same OS
4. **Hardware Virtualization** – Uses CPU support

**Conclusion:**
Each type differs in performance, flexibility, and implementation.

---

## ❓ Q7. Explain Virtualization and Cloud Computing.

**Answer:**
Virtualization is the core technology behind cloud computing.

* It allows multiple virtual machines on a single server
* Cloud providers use virtualization to deliver services

👉 Examples:

* Amazon Web Services
* Microsoft Azure

**Conclusion:**
Without virtualization, cloud computing would not be possible.

---

## ❓ Q8. Write Advantages and Disadvantages of Virtualization.

**Answer:**

### Advantages:

* Efficient resource utilization
* Cost saving
* Scalability
* Isolation

### Disadvantages:

* Performance overhead
* Security risks
* Complex setup

---

Cloud service providers offer cloud-based services to users.

### Services:

# 📘 SECTION C: Long Answer Questions (10–15 Marks)

## ❓ Q10. Explain the process of deploying web services in cloud.

**Answer:**

### Steps:

1. Develop web application
2. Test application
3. Package application
4. Choose cloud provider
5. Deploy on VM or container
6. Configure network (IP, DNS)
7. Monitor and maintain

**Conclusion:**
Cloud deployment ensures scalability and high availability.

---

## ❓ Q11. Explain deploying web services inside and outside cloud architecture.

**Answer:**

### Inside Cloud:

* Hosted on cloud servers
* Uses load balancing and auto scaling

**Advantages:**

* High scalability
* Easy maintenance

**Disadvantages:**

* Cost dependency
* Vendor lock-in

---

### Outside Cloud (On-Premises):

* Hosted on local servers

**Advantages:**

* Full control
* Custom security

**Disadvantages:**

* Expensive
* Limited scalability

---

## ❓ Q12. Explain Cloud Applications with advantages and disadvantages.

**Answer:**

### Cloud Applications:

Applications that run on cloud infrastructure and accessed via internet.

### Advantages:

* Access anywhere
* Cost efficient
* Automatic updates

### Disadvantages:

* Internet dependency
* Security concerns
* Downtime risk
