# 🌐 1. Introduction to IoT (Internet of Things)

**Definition:**
IoT is a system of interconnected physical devices (things) embedded with sensors, software, and network connectivity to collect and exchange data over the internet.

**Example:**
Smart home (AC, lights controlled via mobile), smart watch, smart city systems.

---

# ⭐ 2. Features of IoT

* **Connectivity** → Devices connected via internet
* **Sensing** → Sensors collect real-time data
* **Automation** → Automatic control without human effort
* **Intelligence** → Data analysis for decision-making
* **Scalability** → Can handle millions of devices
* **Real-time processing** → Instant response

# ❌ 5. Disadvantages of IoT

* Security risks (hacking)
* Privacy issues
* High implementation cost
* Dependency on internet
* Complexity in management

# 🏗️ 6. IoT Architecture

### 🔹 3-Layer Architecture

1. **Perception Layer**

   * Sensors & devices
   * Collect data

2. **Network Layer**

   * Transfers data (WiFi, Bluetooth, etc.)

3. **Application Layer**

   * Provides services (apps, dashboards)

---

### 🔹 5-Layer Architecture (Advanced)

1. Perception Layer
2. Transport Layer
3. Processing Layer
4. Application Layer
5. Business Layer

# 📶 17. IoT Communication Layers

### 🔹 Data Link Layer

* Responsible for node-to-node data transfer
* Technologies: WiFi, Bluetooth

### 🔹 Network Layer

* Routing and addressing
* Protocols: IP, IPv6

### 🔹 Session Layer

* Manages connection between devices
* Ensures communication session
