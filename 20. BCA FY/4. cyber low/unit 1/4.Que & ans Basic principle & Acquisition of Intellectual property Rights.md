## Question 1: Explain Philosophical Aspects of Intellectual Property Laws
Writing

Intellectual Property Rights (IPR) are based on philosophical theories.

✓ Natural Rights Theory
• Creator owns what they create
Example → Software developer owns program

✓ Reward Theory
• Creators get reward for work
Example → Scientist invents medicine

✓ Incentive Theory
• Encourages innovation
Example → Developers create apps

✓ Economic Growth Theory
• Innovation improves economy

✓ Social Benefit Theory
• Society benefits from inventions

Conclusion:
These theories justify Intellectual Property Rights.

## Question 2: Explain Basic Principles of Patent Law
Writing

✓ Patent → Legal right given to inventor

### Requirements:

• Novelty → Must be new
• Utility → Must be useful
• Non-obvious → Must be unique

✓ Patent Validity → 20 years

Examples:

• New machine
• New software
• Medical device
