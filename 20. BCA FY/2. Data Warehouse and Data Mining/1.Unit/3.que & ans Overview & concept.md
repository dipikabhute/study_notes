# ⭐ 1. What is Data Warehousing? Explain its Need.

## Answer:

Data Warehouse is a centralized repository used to store integrated, historical, and subject-oriented data for decision making.

### Need for Data Warehousing

### ► 📊 Better Decision Making
Helps management take strategic decisions based on analyzed data.

### ► 📈 Historical Data Analysis
Stores past data to identify trends and patterns.

### ► 🔗 Data Integration
Combines data from multiple sources into one place.

### ► ⚡ Faster Query Processing
Optimized for reporting and analysis.

### ► 🎯 Business Intelligence Support
Supports dashboards and reporting tools.

# ⭐ 4. What is Data Mart?

## Answer:

A Data Mart is a subset of Data Warehouse designed for a specific department.

### Types of Data Mart

► Dependent Data Mart
► Independent Data Mart
► Hybrid Data Mart

### Example

► Sales Data Mart
► HR Data Mart
► Finance Data Mart

# ⭐ 5. What is Data Staging?

## Answer:

Data Staging is a temporary area where data is cleaned and transformed before loading into Data Warehouse.

### Steps in Data Staging

► Data Extraction
► Data Cleaning
► Data Transformation
► Data Loading

# ⭐ 6. What is Metadata?

## Answer:

Metadata means data about data.

### Types of Metadata

► Technical Metadata
► Business Metadata
► Operational Metadata

### Example

► Table Name
► Data Type
► Source Information

 # ⭐ 7. Explain Data Warehousing & ERP

## Answer:

ERP (Enterprise Resource Planning) systems generate operational data.

### Relationship

► ERP provides operational data
► Data warehouse stores historical data
► Used for analysis and reporting

### Benefits

► Better decision making
► Performance analysis

# ⭐ 8. Explain Data Warehousing & KM

## Answer:

Knowledge Management (KM) converts data into useful knowledge.

### Relationship

► Data warehouse stores data
► KM converts data into knowledge
► Helps decision making

# ⭐ 9. Explain Data Warehousing & CRM

## Answer:

CRM (Customer Relationship Management) manages customer information.

### Relationship

► CRM collects customer data
► Data warehouse analyzes data
► Improves customer satisfaction
